Release Notes: 2017-02-16
============================

Thanks for upgrading to the latest version of the ALKS CLI!

- Fixes issue where region and format would be removed from `.aws/credentials`
- Fixes crash when entering invalid password
- Fixes issue where not forcing on existing creds output would log the user out
- Selecting 'No' to Save Password will now clear previous password
- On Linux, `.netrc` is now R+W owner only
- Creates `.aws/credentials` if it doesn't exist
